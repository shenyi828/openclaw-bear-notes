# Bear Notes upgrade ideas

## 已适合继续升级的方向

1. **自动判定新建 / 追加**
   - 先按标题、关键词、向量检索找相似笔记
   - 若高度相似，优先提示是否追加

2. **自动标签推荐**
   - 基于既有 Bear 标签树做推荐，减少乱标

3. **模板路由**
   - 概念卡 / 方法论 / SOP / 会议纪要 / 复盘 自动套模板

4. **写入后更新向量库**
   - 写 Bear 成功后，同步导出/索引到本地检索层

5. **半自动导出 PDF**
   - 保留当前 Bear GUI 半自动链路
   - 继续补桌面自动化点击稳定性

6. **标题去重与命名规范**
   - 自动检查是否已存在同标题或近似标题笔记

7. **用户偏好可配置**
   - 默认目录
   - 默认标签
   - 默认是否先本地再同步
   - 默认是否允许双链
