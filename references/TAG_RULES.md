# Bear Notes Tag Rules

## 主规则

1. 标签首先是目录，不是装饰。
2. 先主目录，后子目录，最后才考虑横向标签。
3. 少而准，胜过多而散。
4. 不乱加工具标签，例如 `Bear`。
5. 不长期保留测试标签，例如：`格式测试`、`高亮测试`、`关联测试`。

## 常用主目录

- `#博闻强记/...`：外部知识、理论、概念、框架
- `#me/...`：个人体验、自我观察、成长记录
- `#工作/...` / `#干细胞/...` / `#Boss/...` / `#币圈`：现实任务和项目内容
- `#小说/...`：创作、设定、章节、人物
- `#考研/...` / `#英语` / `#论文`：应试与学术内容

## 横向标签

仅在确有长期检索价值时使用，例如：
- `#知识管理`
- `#方法论`
- `#向量库`

原则：辅助索引，不替代主目录。
