`v0.1.0 - public beta`

## Suggested release summary

First public beta of `bear-notes`, a Bear.app skill for OpenClaw.

Included in this release:
- structured Markdown note rendering
- Bear note creation via macOS automation
- lightweight tag recommendation
- experimental append-to-note workflow
- tag governance and long-term note-writing rules

Known limitations:
- append flow currently relies on Bear front-end UI automation
- behavior may vary with system language, input method, and focus state
- similarity-based append/new-note routing is not fully implemented yet
